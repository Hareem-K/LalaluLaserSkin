# Lalalu Skin & Laser  

A full, production-ready website built for a Calgary-based med spa using React, TypeScript, and Tailwind CSS.
The platform includes a complete suite of client-facing pages and interactive features, providing a seamless user experience from browsing treatments to booking appointments.

The website features:
- A fully designed Home page with hero banners, service highlights, offers, testimonials, and brand showcases
- A Services directory with detailed treatment pages, pricing tiers, and dynamic aftercare instructions
- A Products section showcasing premium skincare items with image galleries
- A Skin Concerns hub, connecting users to recommended treatments
- A Blog with educational posts and a sidebar for easy navigation
- A Contact page with an integrated EmailJS form and embedded Google Maps
- A FAQ section answering common client questions
- A seamless Book Now page integrating Acuity Scheduling for real-time appointments

Note:
This repository is for portfolio and code-review purposes only and is not intended for open-source deployment.
